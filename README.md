🛒 Amazon Clone (HTML & CSS) -

This is my first GitHub project, where I built a simple Amazon homepage clone using HTML and CSS.
The goal of this project was to practice basic web development and layout design.

📌 Features -

a)Amazon-style homepage layout 

b) Navigation bar with logo and search box

c) Product sections

d) Footer similar to Amazon

e) Responsive layout (basic)

🛠️ Built With -

a) HTML5

b) CSS3

🎯 Learning Goals -

 This project helped me learn:

a) Basic HTML structure

b) CSS styling

c) Layout design
